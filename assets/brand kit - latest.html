Visual Language

Five visual principles

These principles turn references into a repeatable GalaxaTech visual language. Use them for website sections, social posts, ads, pitch decks, and AI-generated graphics.

🌑

Dark Spatial Backgrounds

Void Black (#05030F) base. Violet/blue ambient orbs positioned off-centre. Fine grain overlay (3–5%). Never fill large surfaces with loud gradients.

🔡

Bold Editorial Type

One dominant headline or keyword. Layer type with depth only when it improves hierarchy. Use purple emphasis sparingly.

🪟

Glass Layers

Frosted glass panels float over the dark background. 1px border, 20–24px radius, soft purple inner glow, top sheen highlight.

🧿

Structured 3D Metaphors

Single refined 3D metaphor chosen for meaning (maze=problem, stairs=growth). Controlled violet rim light, realistic material depth, clean shadows.

⊞

Grid + Glow + Grain

Perspective wireframe grid at the bottom. Violet glow at intersection points. Fine noise grain overlay. Starfield dots optional.

Content Rules

How content is structured

Every post, slide and section follows the same three rules: right ratio, a metaphor-led visual, and a CTA that converts without being cheap.

Preferred ratios

1:1

Feed posts, profile thumbnails, ads. Most versatile — compresses well across platforms.

4:5

Tall feed posts on Instagram and Facebook. Shows more copy without the scroll.

16:9

Website headers, YouTube thumbnails, presentation slides, ad banners.

Metaphor-led visuals

Object → Meaning

Every 3D object is chosen because it says something. Maze = friction. Staircase = growth. Chess = strategy. Hourglass = urgency. Diamond = value. Brain = intelligence. Never pick an object just because it looks cool.

Subject layering

The giant display word sits behind the top of the object and in front of the base. This occlusion creates natural 3D depth — the object feels like it's really in front of the type.

Reference integrity: Pinterest or competitor visuals are used only to extract principles — mood, hierarchy, lighting, depth, and composition. GalaxaTech assets must never copy external layouts, logos, characters, campaign structures, or exact visual arrangements.

CTA rules

✓ DO
  • One CTA per post. Clear, specific action.
  • Use "Read the caption", "DM us", "Link in bio" — platform-native language.
  • Glass pill button style on post images.
  • Violet gradient on website primary CTAs.
  • Cool Glow accent for secondary actions.
✕ DON'T
  • Multiple CTAs competing on one post.
  • "Click here", "Buy now" — too generic.
  • Flat colour buttons without glow.
  • Small, low-contrast CTA that blends in.
  • All-caps CTA in body text (save for button labels only).
